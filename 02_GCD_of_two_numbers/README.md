## GCD of two numbers

Lets consider 12 16, GCD of 12 and 16 is 4

### Input Format

```
22 33
```

_Constraints_

_1 >= N <= 100_

### Output Format

```
The GDC of 22 and 33 is 11
```

### Sample Input 0

```
12 16
```

### Sample Output 0

```
4
```

### Sample Input 1

```
0 0
```

### Sample Output 1

```
Not Valid
```
